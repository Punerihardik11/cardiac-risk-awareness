# User Input Schema

## Purpose
Defines the data collected from users to estimate cardiac risk and track lifestyle trends.
This is for awareness and risk estimation, not diagnosis.

---

## 1. Basic Demographics
- Age (years)
- Gender (Male / Female / Other)
- Height (cm)
- Weight (kg)

---

## 2. Vital Health Indicators
- Resting heart rate (bpm)
- Blood pressure (if known)
  - Systolic
  - Diastolic
- Blood sugar level (optional)
- Cholesterol level (optional)

---

## 3. Lifestyle Factors
- Smoking status (Never / Former / Current)
- Alcohol consumption (None / Occasional / Regular)
- Physical activity (minutes per day)
- Average steps per day (manual or device-based)
- Sleep duration (hours)

---

## 4. Symptoms (Self-reported)
- Chest pain (Yes / No)
- Shortness of breath (Yes / No)
- Fatigue (Yes / No)
- Palpitations (Yes / No)
- Dizziness (Yes / No)

---

## 5. Medical History
- Family history of heart disease (Yes / No)
- Existing conditions (Diabetes, Hypertension, etc.)
- Current medications (optional)

---

## 6. Stress & Mental Load
- Perceived stress level (Low / Medium / High)
- Work hours per day
- Screen time per day

---

## Notes
- All inputs are user-reported
- Missing data is allowed
- No emergency or diagnostic use
